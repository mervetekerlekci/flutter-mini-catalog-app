# Flutter Mini Catalog App

A simple Flutter e-commerce catalog application developed with Flutter and Dart.

## Features

- Product listing
- Product detail page
- Search functionality
- Favorites system
- Cart system
- API integration
- Responsive GridView UI

## Technologies

- Flutter
- Dart
- DummyJSON API

## Flutter Version

Flutter 3.41.9

## Run Project

```bash
flutter pub get
flutter run
```
